# lein-as-resource

Provides a dependency containing a lein uberjar.  See
[alembic](https://github.comp/pallet/alembic) for why this might be useful.

## Usage

```clj
[lein-as-resource "2.1.3"]
```

The lein uberjar is provided as "lein-standalone.jar".
